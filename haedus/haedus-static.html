Drop down ask box

Make sure you’ve enabled asks! Go to the appearance section, scroll down until you see a text box labeled “Your tumblr url”. Replace “example.tumblr.com” with your tumblr url without quotations. Save. Help?

Google fonts

You can choose any font you like from Google’s web fonts. You only need to write the font name in the text box called “Google title font” for the blog title, or in the box called “Google body font” for the post body font. Be sure to write the correct name and it works best if you use capitalized letters where there should be. No, the fonts won't show on the preview. That's the way the tumblr preview work, sorry

Social icons

In the appearance section, there are text boxes labeled for different social sites. Just type your username for that specific site in the respective text box and the icon should appear. No, the icons won't show on the preview. That's the way the tumblr preview work, sorry. Quick example: How to add the Instagram icon.

  1. Find the text box labeled "Instagram username"
  2. Type your Intagram username in the box
  3. Press save and enjoy your awesome icon!

Add Disqus

You need to have a Disqus forum in order for this feature to work.

  1. Head to the Disqus site and sign up and/or create a forum
  2. Remember the unique Disqus URL (which is what I call forum name)
  3. Go to the appearance section, and find the text box labeled "Disqus forum name"
  4. Type your forum name in the box
  5. Press save and enjoy your Disqus section!
