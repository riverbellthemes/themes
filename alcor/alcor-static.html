→ Download this theme

A very versitile theme with a fixed sidebar and two post widths in two different layouts. Don’t forget to like or reblog if you are/plan to use this theme! ♥

Features

  • 250/400px posts, single column or multiple columns (as many as the browser window fits)
  • 400px wide and 100% tall sidebar
  • Built in pop up boxes for about section, links and ask box
  • Search bar
  • 10 custom links and page links
  • Option to make background/sidebar image cover or pattern. The background can also be just one picture in either top or bottom right corner
  • Optional infinite scrolling
  • Option to hide captions
  • Optional grayscale images (will regain color when hovered upon)
  • Optional hide tags on permalink page
  • Light or dark reblog + like buttons
  • Three audio player colours
  • Custom fonts from Google fonts (see notes below)
  • Social icons (see notes below)

Rules

Do not steal, redistribute, remove credit or claim as your own. Alterations for personal use are ok!

Custom ask box

Make sure you’ve enabled asks! Go to the appearance section, scroll down until you see a text box labeled “Your tumblr url”. Replace “example.tumblr.com” with your tumblr url without quotations. Save. Help?

Google fonts

You can choose any font you like from Google’s web fonts. You only need to write the font name in the text box called “Google title font” for the blog title, or in the box called “Google body font” for the post body font. Be sure to write the correct name and it works best if you use capitalized letters where there should be. No, the fonts won’t show on the preview. That’s the way the tumblr preview work, sorry

Social icons

In the appearance section, there are text boxes labeled for different social sites. Just type your username for that specific site in the respective text box and the icon should appear. No, the icons won’t show on the preview. That’s the way the tumblr preview work, sorry. Quick example: How to add the Instagram icon.

  1. Find the text box labeled “Instagram username”
  2. Type your Intagram username in the box
  3. Press save and enjoy your awesome icon!

Blog title

In this theme it’s not your usual tumblr blog title that shows up in the sidebar, you’ll have to add a new (or the same if you want) title in the text box labeled “Blog title” in the appearance section.

If your blog preview looks weird

That’s because it just does sometimes. Sometimes I have to choose another “Layout”, save and refresh the page and then it works. But don’t worry about the tumblr preview really, if it looks good on your real blog, it’s fine.
