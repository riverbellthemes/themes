Google fonts

You can choose any font you like from Google’s web fonts. You only need to write the font name in the text box called “Google body font” for the post body font. Be sure to write the correct name and it works best if you use capitalized letters where there should be. No, some fonts won’t show on the preview. That’s the way the tumblr preview work, sorry.

Social icons

In the appearance section, there are text boxes labeled for different social sites. Just type your username for that specific site in the respective text box and the icon should appear. No, the icons might not show on the preview. That’s the way the tumblr preview work, sorry. Quick example: How to add the Instagram icon.

  1. Find the text box labeled “Instagram username”
  2. Type your Intagram username in the box
  3. Press save and enjoy your awesome icon!
