Theme #7 Tinfoil [revamped] by Riverbell themes

A fresh and crisp makeover of my seventh theme, Tinfoil. The inspiration was minimalism and the cold colour of gold, as seen in the preview. This theme is packed with options and cool features which I hope you will enjoy. Don't forget to like or reblog this post if you are thinking of using this theme ♥

Features

  • 400/350/330/300/250 px post width in one column
  • Two sidebars
  • Blog background selections: colour, gradient, full image, pattern image, one image in top right corner, one image in bottom right corner
  • Sidebar background selections: colour, gradient, full image
  • Circular tumblr icon
  • Up to 10 custom links + page link support
  • Optional infinite scroll
  • Optional greyscale posts
  • Optional show tags on hover
  • Custom Google fonts (see notes)
  • Custom reblog and like buttons
  • Disqus support (see notes)

Rules

Do not steal, redistribute, remove credit or claim as your own. Alterations for personal use are ok! 

Google fonts

You can choose any font you like from Google’s web fonts. You only need to write the font name in the text box called “Google title font” for the blog title, or in the box called “Google body font” for the post body font.

Be sure to write the correct name and it works best if you use capitalized letters where there should be. No, the fonts won't show on the preview. That's the way the tumblr preview work, sorry

If you want to be able to see italic and bold fonts on your blog, be sure to use a font that has the styles Normal (400), Normal italic, Bold (700) and Bold Italic.

Recommended body fonts: Open Sans, Roboto, Lato, Alegreya Sans, Source Sans Pro.

Add Disqus

You need to have a Disqus forum in order for this feature to work.

  1. Head to the Disqus site and sign up and/or create a forum
  2. Remember the unique Disqus URL (Disqus shortname)
  3. Go to the appearance section, and find the text box labeled "Disqus shortname"
  4. Type your Disqus name in the box
  5. Press save and enjoy your Disqus section!
